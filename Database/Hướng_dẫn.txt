Để xem database:
B1: Download file sql trong folder
B2: Download mysql, mysql workbench va cài đặt
B3: Tạo một database
B4: Chạy câu lệnh sau trong database nếu chưa có qldiem_csdl

    CREATE DATABASE IF NOT EXISTS qldiem_csdl;
    USE qldiem_csdl;

B5: Sau khi tạo xong, ở phía góc trên bên trái, nhấn vào phần Server rồi Data Import
B6: Chọn Import from self-contained file rồi chọn file sql đã lưu ở B1.
B7: Chọn Start Import
